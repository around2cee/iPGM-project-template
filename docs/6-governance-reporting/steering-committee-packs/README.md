# Steering Committee Packs
> iPGM 6.4 — SteerCo Packs

Store dated steering committee packs in this folder.

## Naming Convention
```
YYYY-MM-DD-steerco-pack.md
```

## What Each SteerCo Pack Should Cover
A concise, decision-focused briefing. Different from status reports —
this brings executives who are not in the day-to-day up to speed and seeks approvals.

Structure:
1. **Program Summary** — 1-line status + phase
2. **Since Last SteerCo** — what happened, what changed
3. **Plan to Next SteerCo** — what we are committing to
4. **Risks Requiring Awareness** — top 1-3
5. **Decisions Required** — specific asks with options and recommendations
6. **Appendix** — milestone tracker, RAID summary, financials if applicable

Keep decks to 5-7 slides / sections. Executives want the headline, not the detail.
