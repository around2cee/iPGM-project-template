# Executive / Leadership Updates
> iPGM 6.3 — Leadership Updates

Store dated executive update documents in this folder.

## Naming Convention
```
YYYY-MM-executive-update.md
```

## What Each Executive Update Should Cover
- Program name and reporting period
- Overall status (1-line headline)
- 3-5 bullet accomplishments
- Key risks or items requiring leadership awareness
- Next major milestone
- Any asks or decisions required from leadership

Executive updates are less frequent than status reports and are focused on
providing leadership visibility — not asking for decisions (though that can happen in read-outs).
