# 2.7 Assumptions Register
> **PGM Role:** Creator & Owner | **Phase:** Planning (optional)

A living list of statements believed to be true that the program relies on —
but which are not yet proven or guaranteed.

Answers: "What are we implicitly counting on being true for this program to succeed?"

Different from:
- **Risks** — something that might go wrong
- **Dependencies** — something that must happen
- **Constraints** — a fixed limitation
