# 5.5 Decision Register / Change Log
> **PGM Role:** Creator & Owner | **Phase:** Initiation → Closure (ongoing)

Formal record of all program decisions and approved changes to the baseline.
The PGM is the historian of the program — this is how that history is captured.

Use the Decision / Change Log GitHub Issue Template to create traceable entries.

---
