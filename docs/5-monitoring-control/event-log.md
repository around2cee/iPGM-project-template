# 5.6 Event Log
> **PGM Role:** Creator & Owner | **Phase:** Initiation → Closure (ongoing)

Consolidated chronological log of all significant events in this program.
Provides an accurate representation of the program timeline for the Closure Report.

Includes: decisions, missed deadlines, risks that materialized, scope changes,
escalations, milestone completions, go/no-go calls.

---

## Event Log

| Date | Event Type | Description | Impact | Owner / Reported By | Reference |
|---|---|---|---|---|---|
| YYYY-MM-DD | Decision / Risk / Milestone / Change / Escalation | | None / Minor / Major | | Issue # / Doc link |

---

_Log events as they happen — recreating this after the fact is difficult and error-prone._
_This discipline is what makes the Program Closure Report credible._
