# Meeting Notes
> iPGM 6.5 — Program Meeting Notes

Store all program meeting notes in this folder. The PGM is responsible for
facilitating meetings and ensuring notes are captured and published here.

Do NOT use email for meeting notes. Use email / Slack only to notify stakeholders
of where to find the notes (link to this folder).

## Naming Convention
```
YYYY-MM-DD-[meeting-type]-notes.md

Examples:
  2026-04-07-core-team-sync-notes.md
  2026-04-14-sponsor-checkin-notes.md
  2026-04-21-steerco-notes.md
```

## Standard Meeting Notes Structure

```markdown
# [Meeting Type] — YYYY-MM-DD

**Attendees:** Name (Role), Name (Role)
**Facilitator:** Name
**Notes by:** Name

## Agenda
1.
2.

## Discussion & Decisions
- [Topic]: [summary of discussion and outcome]

## Action Items
| Action | Owner | Due Date |
|---|---|---|
| | | |

## Next Meeting
Date: | Agenda items to carry forward:
```
